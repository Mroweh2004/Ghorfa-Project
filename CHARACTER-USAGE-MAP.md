## 📱 Responsive Behavior

All characters automatically resize based on screen width:

- **Desktop (>1024px):** Full size
- **Tablet (768-1024px):** 80-85% size
- **Mobile (<768px):** 60-70% size

Characters also reposition on mobile:
- Header characters → move below text
- Sidebar characters → remain centered
- Empty state characters → remain centered
